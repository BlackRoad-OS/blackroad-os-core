# 🛣️ BlackRoad Terminal Dashboard & Agents

This is a tiny playground for:
- 🎛️ A terminal UI dashboard (HTTP, network, system views)
- 🤖 Local "agents" that help you run checks
- 🧪 Safe experiments in `~/blackroad-sandbox`

Nothing here should touch your real system except:
- Reading basic info (uptime, disk usage, IP)
- Curling health-check URLs you configure
