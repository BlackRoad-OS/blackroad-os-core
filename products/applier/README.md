# 🚗 applier - Job Application Automation

**Stop applying to jobs manually. Let AI do it for you.**

## What This Does

`applier` is a command-line tool that automates job applications across multiple platforms:

- LinkedIn
- Indeed
- Glassdoor
- Monster
- And 30+ more platforms

## Quick Start

```bash
# Install (requires Python 3.8+)
python3 applier-cli.py

# Run interactive setup
./applier-cli.py

# Start applying to jobs
applier apply --role "Software Engineer" --count 10
```

## Features

✅ AI-powered resume customization
✅ Cover letter generation
✅ Multi-platform support (30+)
✅ Application tracking
✅ Daily automation
✅ Email notifications

## Requirements

- Python 3.8 or higher
- Mac or Linux (Windows WSL works too)
- Terminal/command line access
