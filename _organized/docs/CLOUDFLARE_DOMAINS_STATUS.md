### For Domains You Own

2. Click "Add a Site"
3. Enter domain name
4. Follow nameserver instructions
5. Update nameservers at your registrar

- Google Domains

## 📋 Custom Domain Configuration

Once domains are in Cloudflare, add custom domains:

### Method 1: Via Dashboard (Easiest)

1. https://dash.cloudflare.com → Workers & Pages
2. Click project name
3. Custom domains → Set up a custom domain
4. Enter domain → Continue

### Method 2: Via CLI

```bash
# Example
wrangler pages domain add blackroad.systems --project-name=blackroad-systems
wrangler pages domain add portal.blackroad.systems --project-name=blackroad-systems
```
