## 🚀 DEPLOYMENT STEPS

### 1. Prerequisites (3 minutes)

```bash
# Install CLIs
brew install gh
npm install -g @railway/cli wrangler

# Authenticate
gh auth login
railway login
wrangler login
```

### 2. Deploy All Services (30-45 minutes)

```bash
cd ~/blackroad-services-phase1
./MASTER_DEPLOYMENT_AUTOMATION.sh
```

### 3. Configure Secrets (5 minutes)

```bash
# Get tokens
railway whoami    # Copy RAILWAY_TOKEN
wrangler whoami   # Copy CLOUDFLARE_API_TOKEN and ACCOUNT_ID

# Add to all GitHub repos (automated)
for repo in $(gh repo list BlackRoad-Services --limit 100 --json name -q '.[].name'); do
  gh secret set RAILWAY_TOKEN --body="$RAILWAY_TOKEN" --repo="BlackRoad-Services/$repo"
  gh secret set CLOUDFLARE_API_TOKEN --body="$CF_TOKEN" --repo="BlackRoad-Services/$repo"
  gh secret set CLOUDFLARE_ACCOUNT_ID --body="$CF_ACCOUNT" --repo="BlackRoad-Services/$repo"
done
```

### 4. Configure Custom Domains (10 minutes)

```bash
# Add CNAME records in Cloudflare DNS
# Example: identity.blackroad.io → blackroad-identity.pages.dev
```

### 5. Enable OIDC & Mesh (15 minutes)

```bash
# Create OIDC clients in blackroad-identity (Keycloak)
# Register services with blackroad-mesh (Headscale)
```
