### To Deploy to Homebrew:

1. Create `BlackRoad-OS/homebrew-tap` repo on GitHub
2. Upload `Formula/lucidia.rb`
3. Tag release `v1.0.0` on `blackroad-os-core`
4. Update SHA256 in formula
5. `brew tap BlackRoad-OS/tap`
6. `brew install lucidia`

### To Deploy to PyPI:

```bash
cd lucidia
python3 -m build
twine upload dist/*
```

Then anyone can: `pip install lucidia`
