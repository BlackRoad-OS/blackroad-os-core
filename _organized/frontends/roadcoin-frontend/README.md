# RoadChain Frontend

Beautiful Next.js frontend for roadchain.io

## Features

- Live Lucidia breath indicator
- Blockchain explorer
- Technical specifications
- Genesis block visualization
- Responsive design

## Deploy

```bash
pnpm install
pnpm build
npx wrangler pages deploy out --project-name=roadchain-io
```
