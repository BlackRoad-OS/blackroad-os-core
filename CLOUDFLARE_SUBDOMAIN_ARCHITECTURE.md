## Dynamic App Architecture

### Each Subdomain Gets:

1. **Cloudflare Worker** - Dynamic routing
2. **Pages Function** - Server-side logic
3. **KV Storage** - Fast key-value storage
4. **D1 Database** - SQL database (shared or dedicated)
5. **R2 Storage** - Object storage for assets
6. **SSL/TLS** - Automatic HTTPS
7. **Analytics** - Built-in analytics
8. **Caching** - Edge caching

### Worker Template Structure

```typescript
export default {
  async fetch(request, env, ctx) {
    const url = new URL(request.url);
    const subdomain = url.hostname.split('.')[0];

    // Route based on subdomain
    const app = SUBDOMAIN_APPS[subdomain];

    if (!app) {
      return new Response('App not found', { status: 404 });
    }

    return app.handleRequest(request, env, ctx);
  }
}
```
